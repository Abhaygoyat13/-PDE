 PDE Policy – Key Points
Intent-Based Control: All requests must be structured and validated before execution.
Domain Restrictions: Only approved domains allowed; sensitive sectors are automatically blocked.
Operational Limits: Analysis depth, problem count, and actions are strictly constrained.
Risk Evaluation: Every request is risk-scored; high-risk inputs are denied.
Delegation Boundaries: Sub-agents operate under limited, non-expandable permissions.
Transparency: All decisions are logged and blocked actions are explained.
Prohibited Uses: No sensitive intelligence, manipulation, unauthorized data use, or scope expansion.
